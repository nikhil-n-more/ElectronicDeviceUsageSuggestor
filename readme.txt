Instructions on using the web application

1) Open the project folder where manage.py file is located
2) run "python manage.py runserver" command to start the server
3) Open the "localhost:8000" in web browser
4) Read the instructions and click proceed
5) Enter the convinient information and click next
6) At last submit and algorithm will calculate the suggesstions
7) Choose the best alternative among the suggested ones
